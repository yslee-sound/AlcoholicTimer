## 🔍 노출(Impression) vs 클릭(Click)

### 광고 노출 (Impression)

**정의**:
- 광고가 사용자 화면에 표시되는 것
- 광고가 일정 비율(보통 50%) 이상 보이는 것
- 일정 시간(보통 1초) 이상 화면에 머무는 것

**수익**:
- 노출당 수익: $0.001 ~ $0.01 (매우 낮음)
- 1,000회 노출 = $1 ~ $10 (CPM 기준)

**정책**:
- ✅ **본인이 보는 것 완전히 허용**
- ✅ 자연스러운 앱 사용 중 노출
- ✅ 정상적인 수익 발생

---

### 광고 클릭 (Click)

**정의**:
- 광고를 손가락으로 터치하는 것
- 광고주 사이트로 이동하는 것
- 광고 상세 페이지 보는 것

**수익**:
- 클릭당 수익: $0.10 ~ $2.00 (매우 높음)
- 10회 클릭 = $1 ~ $20 (CPC 기준)

**정책**:
- ❌ **본인이 클릭하는 것 절대 금지**
- ❌ 부정 클릭으로 간주
- ❌ 계정 정지 위험

## 📈 AdMob 수익 구조

### 노출 수익 (Impression-based)

**CPM (Cost Per Mille)**: 1,000회 노출당 수익
```
국가별 CPM:
- 한국: $1 ~ $5
- 미국: $5 ~ $15
- 인도: $0.5 ~ $2

계산:
1,000회 노출 × $3 (평균 CPM) = $3
개발자가 10번 봄 = 10회 노출 = $0.03
```

**특징**:
- ✅ 안정적인 수익
- ✅ 개발자가 봐도 정상
- ✅ 예측 가능
- ❌ 수익이 낮음

---

### 클릭 수익 (Click-based)

**CPC (Cost Per Click)**: 클릭당 수익
```
광고 종류별 CPC:
- 게임: $0.10 ~ $0.50
- 금융: $1.00 ~ $3.00
- 쇼핑: $0.20 ~ $1.00

계산:
100회 노출 × 5% CTR = 5회 클릭
5회 클릭 × $0.50 (평균 CPC) = $2.50
```

**특징**:
- ✅ 높은 수익
- ❌ 개발자 클릭 금지
- ❌ 부정 클릭 감지
- ❌ 계정 정지 위험

## 🛡️ AdMob 부정 클릭 감지 시스템

### 감지 방법

**개발자 본인 클릭 감지**:
1. **기기 ID 추적**
   - 앱 설치 기기 = 광고 클릭 기기
   - 개발자 계정과 연결된 기기

2. **IP 주소 추적**
   - 광고 클릭 IP = 앱 업로드 IP
   - 같은 네트워크에서 반복 클릭

3. **패턴 분석**
   - 같은 시간대 반복 클릭
   - 광고 표시 직후 즉시 클릭
   - 비정상적으로 높은 CTR

4. **계정 연결**
   - Google 계정으로 로그인
   - Play Console 계정과 연결
   - AdMob 계정과 연결

**판정 기준**:
```
정상 사용자:
- CTR: 1~5%
- 클릭 패턴: 무작위
- 체류 시간: 다양함

개발자 클릭 (의심):
- CTR: 50~100%
- 클릭 패턴: 규칙적
- 체류 시간: 매우 짧음
```

---

### 처벌 단계

**1단계: 경고**
```
상황: 1~3회 의심 클릭
조치: 이메일 경고
영향: 없음
```

**2단계: 수익 보류**
```
상황: 반복적인 의심 클릭
조치: 수익 지급 보류
영향: 수익 손실
```

**3단계: 계정 정지**
```
상황: 명백한 부정 클릭
조치: 계정 영구 정지
영향: 
- 모든 앱의 광고 중단
- 누적 수익 몰수
- 재가입 불가
- 복구 불가능
```
