# 1초 애니메이션 테스트 설정 완료

**작업일**: 2025-12-17  
**목적**: 슬라이드 업/다운 효과를 명확하게 확인하기 위한 테스트

---

## ✅ 변경 사항

### 애니메이션 타이밍 조정

**Before (400ms):**
```kotlin
enter: 400ms
exit: 350ms
```

**After (1000ms - 테스트용):**
```kotlin
enter: 1000ms (1초)
exit: 1000ms (1초)
```

---

## 🎯 설정된 애니메이션

### 열기 (SlideInVertically)
```
0ms:     화면 하단 (100%)
250ms:   25% 올라옴
500ms:   50% 올라옴 (중간)
750ms:   75% 올라옴
1000ms:  완전히 표시 ✅
```

### 닫기 (SlideOutVertically)
```
0ms:     완전히 표시
250ms:   25% 내려감
500ms:   50% 내려감 (중간)
750ms:   75% 내려감
1000ms:  화면 하단 (완전히 사라짐) ✅
```

---

## 📝 테스트 방법

### 1. 열기 테스트
1. Tab 4 (커뮤니티) 이동
2. 상단 입력 박스 클릭
3. **관찰**: 
   - 화면 하단에서 시작
   - **1초 동안** 천천히 올라옴
   - 슬라이드 업 효과 명확히 보임 ✅

### 2. 닫기 테스트
1. 글쓰기 창에서 X 버튼 클릭
2. **관찰**:
   - **1초 동안** 천천히 내려감
   - 슬라이드 다운 효과 명확히 보임 ✅

---

## 💡 예상 결과

### 이제 확인 가능한 것들
1. ✅ **슬라이드 업**: 화면 하단 → 상단으로 1초 동안 이동
2. ✅ **슬라이드 다운**: 화면 상단 → 하단으로 1초 동안 이동
3. ✅ **FastOutSlowInEasing**: 시작 빠름 → 끝 천천히
4. ✅ **부드러운 전환**: 각 단계가 눈에 보임

## 🔧 적용된 코드

```kotlin
AnimatedVisibility(
    visible = visible,
    enter = slideInVertically(
        initialOffsetY = { it },
        animationSpec = tween(
            durationMillis = 1000, // ← 1초 (테스트용)
            easing = FastOutSlowInEasing
        )
    ),
    exit = slideOutVertically(
        targetOffsetY = { it },
        animationSpec = tween(
            durationMillis = 1000, // ← 1초 (테스트용)
            easing = FastOutSlowInEasing
        )
    )
)
```
