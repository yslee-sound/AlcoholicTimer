## 🎯 페이스북 스타일 타이밍

### 왜 열기와 닫기가 다를까요?

**열기 (400ms) - 조금 더 여유있게**
- 사용자가 **새로운 콘텐츠를 기다리는** 상황
- 무슨 일이 일어나는지 명확히 보여줌
- 부드럽고 프리미엄한 느낌

**닫기 (350ms) - 조금 더 빠르게**
- 사용자가 **빨리 돌아가고 싶어하는** 상황
- 반응이 빠르다는 느낌
- 대기 시간 최소화

## ✅ 최종 적용된 설정

```kotlin
// 열기 애니메이션
enter = slideInVertically(
    animationSpec = tween(
        durationMillis = 400, // 페이스북 스타일
        easing = FastOutSlowInEasing
    )
)

// 닫기 애니메이션
exit = slideOutVertically(
    animationSpec = tween(
        durationMillis = 350, // 페이스북 스타일
        easing = FastOutSlowInEasing
    )
)

// LaunchedEffect delay
delay(350) // 닫기 애니메이션과 동일
```
