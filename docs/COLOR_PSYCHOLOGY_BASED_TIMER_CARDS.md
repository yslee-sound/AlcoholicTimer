## 🎨 색채 심리학 기반 색상 매칭

### 1. 금주 (Alcohol) → 🌊 깊은 블루
```kotlin
Color(0xFF3B82F6) → Color(0xFF1D4ED8)  // Deep Blue → Royal Blue
```

**심리학적 의미:**
- 💧 **맑은 물:** 술의 혼탁함에서 벗어남
- 🧠 **깨끗한 정신:** 이성적 판단, 자제력
- 🌊 **정화:** 투명하고 깨끗한 상태

**왜 이 블루인가?**
- 진하고 선명한 블루로 가독성 확보
- "술이 없는 맑은 아침"을 시각적으로 표현
- 이전 2번 카드와 동일한 색상으로 검증된 색상

---

### 2. 금연 (Smoking) → 🌿 치유의 그린
```kotlin
Color(0xFF10B981) → Color(0xFF14B8A6)  // Emerald Green → Teal
```

**심리학적 의미:**
- 🫁 **폐의 정화:** 담배 연기로부터 회복
- 🌱 **재생과 치유:** 건강한 호흡
- 💚 **생명력:** 자연의 치유력

**왜 그린인가?**
- 녹색은 가장 강력한 치유와 건강의 색
- "건강한 숨, 깨끗한 폐"를 직관적으로 전달

---

### 3. 사용자 정의 → 🔮 신비로운 퍼플
```kotlin
Color(0xFF8B5CF6) → Color(0xFF7C3AED)  // Vivid Purple → Deep Purple
```

**심리학적 의미:**
- 🎯 **다양한 목표:** 도파민, 카페인, 게임, SNS 등
- ✨ **고급스러움:** 특별함, 개성
- 🧘 **집중:** 명상, 자기 계발

**왜 퍼플인가?**
- 중립적이면서도 세련된 느낌
- 다양한 목표를 포용하는 포괄적 색상
