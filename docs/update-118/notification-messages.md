> "**리텐션 알림의 문구를 구체화하고 다국어(i18n)를 적용하려고 해. 아래 가이드에 따라 작업을 진행해 줘.**
> 1. **리소스 파일 구조 생성:**
> * `res/values/strings.xml` (기본/영어)
> * `res/values-ko/strings.xml` (한국어)
> * `res/values-ja/strings.xml` (일본어)
> * `res/values-in/strings.xml` (인도네시아어)
>
>
> 2. **String Key 정의 및 내용 추가:**
     > 아래 내용을 각 언어별 `strings.xml`에 정확히 추가해.
     > **[Group A Keys]**
> * `notif_group_a_1_title`, `notif_group_a_1_body` (24시간 내용)

> * `notif_group_a_3_title`, `notif_group_a_3_body` (4일 후 내용)
>
>
> **[Group B Keys]**
> * `notif_group_b_3day_title`, `notif_group_b_3day_body` (3일차 내용)

> * `notif_group_b_30day_title`, `notif_group_b_30day_body` (30일차 내용)
>
>
> **[Group C Keys]**
