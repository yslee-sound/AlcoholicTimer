## ⚠️ 중요: 본인 광고 클릭 금지 정책

### 🚨 절대 금지 사항

#### 본인 앱의 광고를 클릭하면 안 됩니다!

**위반 시 결과**:
- 🔴 **AdMob 계정 영구 정지** (무효 트래픽)
- 🔴 **수익 몰수** (지급 보류 또는 환수)
- 🔴 **Google Play 개발자 계정 제재 가능**

### ❌ 금지 행위

1. **본인이 직접 클릭**
   - 자신의 폰에 설치 후 광고 클릭
   - 에뮬레이터/테스트 기기에서 실제 광고 클릭
   - "테스트 목적"이라도 실제 광고는 절대 클릭 금지

2. **지인에게 클릭 요청**
   - 가족, 친구에게 광고 클릭 부탁
   - "한 번만 클릭해 달라" 요청

3. **인위적 조작**
   - 클릭 교환 (서로 앱 깔고 클릭)
   - 자동 클릭 봇/스크립트 사용
   - VPN으로 IP 바꿔가며 클릭

### ✅ 안전한 테스트 방법

#### 1. 디버그 빌드 사용 (권장)
```bash
# 디버그 빌드 설치 - 테스트 광고만 표시됨
gradlew installDebug
```
- Google 테스트 광고만 표시
- **클릭해도 안전** (수익 발생 안 함)
- 광고 레이아웃/동작 테스트 가능

#### 2. 테스트 기기 등록
`MainApplication.kt`에 이미 설정되어 있음:
```kotlin
val testDeviceIds = listOf(
    "33BE2250B43518CCDA7DE426D04EE231" // 테스트 기기 ID
)
```

**본인 기기를 테스트 기기로 추가하는 방법**:
1. 릴리즈 빌드를 기기에 설치
2. 앱 실행 후 Logcat에서 테스트 기기 ID 확인:
   ```
   Use RequestConfiguration.Builder().setTestDeviceIds(Arrays.asList("YOUR_DEVICE_ID"))
   ```
3. 해당 ID를 `MainApplication.kt`에 추가
4. 테스트 기기에서는 실제 광고 대신 테스트 광고 표시됨

#### 3. AdMob 광고 검토 도구 사용
- AdMob 콘솔 > 앱 > 광고 유닛 > "테스트 기기" 설정
- 특정 기기를 테스트 모드로 등록

### 📱 본인 폰에서 안전하게 테스트하는 방법

```bash
# 1. 디버그 빌드 설치 (테스트 광고만 표시)
gradlew installDebug

# 2. 앱 기능 테스트
# - 광고 위치 확인
# - 광고 로딩 확인
# - 레이아웃 확인
# ✅ 테스트 광고는 클릭해도 안전

# 3. 릴리즈 빌드는 다른 사람에게 테스트 요청
# 또는 본인 기기를 테스트 기기로 등록
```

### 🔍 Google의 무효 트래픽 감지

Google은 다음을 모두 추적합니다:
- IP 주소
- 기기 ID (GAID)
- 클릭 패턴
- 개발자 계정과의 연관성
- GPS 위치

**본인이 클릭하면 100% 감지됩니다!**

### 📊 AdMob 콘솔 모니터링

**정상적인 지표**:
- CTR (클릭률): 1-5% 정도
- 다양한 지역/기기에서 고른 분포
- 자연스러운 클릭 패턴

**의심스러운 지표** (자동 검토 대상):
- 비정상적으로 높은 CTR (10% 이상)
- 특정 IP/기기에서 반복 클릭
- 개발자와 동일한 IP에서 클릭

### 💡 올바른 수익화 방법

1. **자연스러운 사용자 유입**
   - Play Store 검색 최적화 (ASO)
   - 실제 사용자에게 유용한 기능 제공
   - 자연스러운 다운로드 증가

2. **광고 최적화**
   - 적절한 광고 빈도 (현재 정책 준수 중 ✅)
   - 사용자 경험을 해치지 않는 배치
   - 높은 앱 품질 유지 → 자연스러운 CTR 상승

3. **장기적 관점**
   - 초기에는 수익이 적음 (정상)
   - 사용자 수 증가에 따라 자연스럽게 증가
   - 무효 트래픽 없이 안정적 운영

### ⚡ 정책 위반 시 대응

만약 실수로 클릭했다면:
1. 즉시 중단
2. AdMob 지원팀에 사전 보고 (선택)
3. 이후 절대 클릭하지 않음
4. 테스트 기기로 등록하여 예방

**한 번의 실수**는 이해될 수 있지만, **반복되면 계정 정지**입니다.
