### ✅ 제목: **알중시계 앱 디자인 가이드 완전판**
- Jetpack Compose 기반으로 구현하며, 미니멀하고 직관적인 UX를 제공합니다.
- 금주 설정 화면에서 검증된 디자인 철학을 전체 앱에 적용합니다.

---

## 🎨 1. 디자인 철학 & 핵심 컨셉

| ----------- | ------------------------------------- |

## 🎨 2. 컬러 시스템 (완성판)

### **기본 컬러 팔레트**
| 용도        | 색상 코드              | 사용 예시                  | 느낌           |
| --------- | ------------------ | ---------------------- | ------------ |
| **Primary** | `#FFFFFF`          | 전체 배경, 카드 배경          | 깨끗함, 순수함    |
| **Text**    | `#000000`          | 제목, 본문, 버튼 텍스트        | 명확함, 가독성    |
| **Border**  | `#000000` (두께 조절)  | 버튼 테두리, 카드 테두리, 구분선   | 절제됨, 구조화    |
| **Accent**  | `#EEEEEE`          | 비활성화 상태, 미달성 레벨       | 차분함, 대기 상태  |

### **상태별 컬러 적용**
| 상태      | 색상 조합                    | 적용 요소           |
| ------- | ------------------------ | --------------- |
| 활성화     | 검은 텍스트 + 검은 테두리         | 현재 선택된 옵션      |
| 비활성화    | 회색 텍스트 + 연한 회색 테두리     | 미선택/미달성 옵션    |
| 강조      | 굵은 검은 텍스트 + 두꺼운 테두리    | 중요한 정보, 레벨 표시 |

## 📐 3. 타이포그래피 시스템

### **폰트 계층 구조**
| 레벨       | 크기    | 굵기      | 용도                    | 예시               |
| -------- | ----- | ------- | --------------------- | ---------------- |
| **H1**   | 24sp  | Bold    | 화면 제목                | "금주 설정"          |
| **H2**   | 20sp  | Bold    | 섹션 제목                | "음주 비용"          |
| **H3**   | 18sp  | Bold    | 서브 제목, 현재 상태         | "현재 진행도"         |
| **Body1** | 16sp  | Regular | 기본 본문, 옵션 텍스트        | "저(1만원 이하)"      |
| **Body2** | 14sp  | Regular | 보조 설명, 부가 정보         | "다음 레벨까지 12일 남음" |
| **Caption** | 12sp  | Regular | 캡션, 힌트 텍스트          | "예: 30"          |

### **텍스트 정렬 규칙**
- **제목**: 중앙 정렬
- **본문/옵션**: 좌측 정렬
- **상태 정보**: 중앙 정렬

## 🧩 4. 컴포넌트 디자인 가이드

### **버튼 디자인**
```kotlin
// 기본 버튼 스타일
OutlinedButton(
    border = BorderStroke(1.dp, Color.Black),
    colors = ButtonDefaults.outlinedButtonColors(
        containerColor = Color.Transparent,
        contentColor = Color.Black
    )
)
```

### **라디오 버튼 그룹**
```kotlin
// 옵션 선택 UI (금주 설정 화면 스타일)
Row(horizontalArrangement = Arrangement.spacedBy(8.dp)) {
    RadioButton(selected = isSelected, onClick = { })
    Text("옵션명", fontSize = 16.sp)
}
```

### **카드/컨테이너**
```kotlin
// 정보 표시 카드
Card(
    colors = CardDefaults.cardColors(containerColor = Color.White),
    border = BorderStroke(1.dp, Color.Black),
    elevation = CardDefaults.cardElevation(0.dp)
)
```

### **구분선**
```kotlin
// 섹션 구분
HorizontalDivider(
    modifier = Modifier.padding(vertical = 16.dp),
    thickness = 1.dp,
    color = Color.Black
)
```

```kotlin

    shape = CircleShape,
    border = BorderStroke(levelThickness.dp, Color.Black),
    color = if (achieved) Color.White else Color(0xFFEEEEEE)
) {
    Text(levelNumber, modifier = Modifier.padding(16.dp))
}
```

## 🎨 8. 애니메이션 & 인터랙션

### **애니메이션 원칙**

- **미세함**: 버튼 클릭 시 미세한 스케일 변화만
- **자연스러움**: 상태 변화는 fade in/out으로 부드럽게

### **인터랙션 피드백**
```kotlin
// 버튼 클릭 피드백
modifier = Modifier.clickable(
    indication = rememberRipple(color = Color.Black, radius = 24.dp),
    interactionSource = remember { MutableInteractionSource() }
)
```

## 🎯 디자인 컨셉 최종 요약

**"절제된 완벽함으로 본질에 집중하는 금주 도우미 앱"**

1. **시각적 단순함**: 흰색 배경 + 검은색 요소만 사용
2. **기능적 명확함**: 모든 인터페이스가 직관적이고 예측 가능
3. **감정적 안정감**: 여백과 정렬을 통한 심리적 평온함 제공
4. **진전감 표현**: 테두리 두께로 성장과 발전을 시각화

이 가이드를 따라 구현하면 사용자에게 **"깨끗하고 집중된 환경에서 차분하게 회복해 나가는"** 경험을 제공할 수 있습니다.
